## 🏆 Featured Projects

### 🔐 Zero-Knowledge Secure File Vault (InfoSec)
*A secure, end-to-end encrypted file sharing REST API built with **Python & Flask**.*
- Architected sharing and editing pipelines utilizing **AES-256-GCM** for payload encryption and **RSA-OAEP** to securely distribute File Encryption Keys (FEKs).
- Hardened backend endpoints with strict **JWT-based** ownership validation and time-expiring access controls to prevent unauthorized privilege escalation.

### ☁️ STANS Application & DevSecOps Pipeline
*A highly available, containerized web application deployed to **AWS EC2**.*
- Engineered an automated **GitHub Actions CI/CD** pipeline utilizing multi-stage **Docker** builds to minimize image footprint.
- Provisioned a secure production environment with **Nginx** reverse proxying, **Let's Encrypt SSL**, and strict firewall configurations.
- Integrated a real-time observability stack using **Prometheus & Grafana** to monitor host resources and system anomalies.

### ⚙️ Large-Scale CI/CD Automated Testing
*Infrastructure automation for a collaborative open-source repository.*
- Authored and maintained complex **GitHub Actions** workflows to validate code integrity, streamline pull requests, and prevent regression bugs across multiple environments.
